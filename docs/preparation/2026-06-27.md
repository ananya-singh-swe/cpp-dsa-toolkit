## Planned Compilation Settings

C++ programs will use the C++20 standard and strong compiler warnings:

```bash
g++ -std=c++20 \
    -Wall \
    -Wextra \
    -Wpedantic \
    -g \
    source.cpp \
    -o program
```

These options provide:

* C++20 language support
* common and additional warnings
* warnings for non-standard behaviour
* debugging information

## Development Workflow

The planned workflow for each implementation is:

1. Understand the requirement
2. Write the simplest correct approach
3. Compile with warnings enabled
4. Test normal and edge cases
5. Debug incorrect behaviour
6. Analyse time and space complexity
7. Improve the solution when necessary
8. Document the reasoning and mistakes
9. Commit meaningful progress to GitHub
